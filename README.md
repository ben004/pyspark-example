## Run Locally

Clone the project

```bash
  git clone https://github.com/ben004/pyspark-example.git
```

Go to the project directory

```bash
  cd pyspark-example
```

Install dependencies

```bash
python3
```

Start the server

```bash
docker-compose up
```

```bash
spark-submit main.py
```
